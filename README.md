# Discord Бот Верификации

Простой бот для верификации пользователей на Discord сервере с помощью кнопки.

## Установка

1. Установи Python 3.8 или выше
2. Установи зависимости:
```
pip install -r requirements.txt
```

3. Создай бота в [Discord Developer Portal](https://discord.com/developers/applications)
4. Скопируй токен бота и вставь в `config.json`
5. Включи в настройках бота:
   - Privileged Gateway Intents → Server Members Intent
   - Privileged Gateway Intents → Message Content Intent

## Настройка

1. Запусти бота: `python bot.py`
2. Пригласи бота на сервер (нужны права Administrator)
3. Создай роль для верифицированных пользователей
4. Используй команду: `!setrole @роль`
5. Создай канал для верификации
6. Используй команду: `!setup`

## Команды

- `!setrole @роль` - Установить роль для верификации (только админы)
- `!setup` - Создать сообщение с кнопкой верификации (только админы)
- `!help_verify` - Показать список команд

## Как работает

1. Пользователь заходит на сервер
2. Видит сообщение с кнопкой "✅ Верифицироваться"
3. Нажимает кнопку
4. Получает роль и доступ к каналам

## Настройка прав

Рекомендуется:
- Закрыть все каналы от @everyone
- Открыть каналы только для роли верифицированных
- Оставить открытым только канал с верификацией
