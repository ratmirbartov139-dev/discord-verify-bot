# Discord Бот Верификации

Простой бот для верификации пользователей на Discord сервере с помощью кнопки.

## Быстрый старт

1. Установи Python 3.8+ и зависимости:
```bash
pip install discord.py
```

2. Создай бота на [Discord Developer Portal](https://discord.com/developers/applications):
   - New Application → Bot → Reset Token
   - Включи: Server Members Intent и Message Content Intent
   - OAuth2 → URL Generator → bot + Administrator → скопируй ссылку
   - Открой ссылку и добавь бота на сервер

3. Вставь токен в `config.json`:
```json
{
    "bot_token": "ТВОЙ_ТОКЕН_СЮДА",
    "verified_role_id": 0
}
```

4. Запусти бота:
```bash
python bot.py
```

## Настройка на сервере

1. Создай роль "Участник" (или любую другую)
2. Убедись что роль бота ВЫШЕ роли "Участник" в списке ролей
3. В Discord напиши: `!setrole @Участник`
4. В канале верификации напиши: `!setup`
5. Готово! Пользователи нажимают кнопку и получают роль

## Команды

- `!setrole @роль` - Установить роль для верификации (только админы)
- `!setup` - Создать сообщение с кнопкой (только админы)
- `!help_verify` - Показать помощь
